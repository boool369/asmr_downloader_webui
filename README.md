### 🌟 项目简介

**ASMR Downloader WebUI** 是一个基于 **Python** 和 **Gradio** 构建的强大 Web 界面下载工具，专为音频爱好者设计。它将复杂的下载流程转化为直观的、跨平台的 Web 界面操作，让您能轻松地获取 RJ 系列音频作品的文件信息、批量搜索、并高效稳定地下载所需文件。

我们致力于提供**流畅、可视化**的下载体验，所有操作（包括配置、搜索和下载进度）均在浏览器中完成。

### ✨ 核心功能亮点

  * **🌐 浏览器可视化界面 (Gradio)**：无需命令行，所有操作通过精美的 Web 界面完成。
  * **🚀 高效并发下载**：利用 `aiohttp` 实现异步并发，最大化下载速度和稳定性。
  * **🔍 关键词智能搜索**：快速根据关键词、标签搜索作品，支持分页和每页数量设置。
  * **📥 灵活下载控制**：支持单个 RJ ID 文件列表展示、**自动全选**、以及搜索结果的**批量顺序下载**。
  * **⚙️ 增强配置管理**：新增 Web UI 配置项，允许用户在界面上设置**下载代理**、**App 运行地址和端口**等关键参数。
  * **📝 实时日志追踪**：内置日志查看器，实时掌握下载任务的每一个细节。

### 🚀 快速开始

#### 1\. 环境准备

确保您的系统已安装 **Python 3.10** 或更高版本。

#### 2\. 安装依赖

打开命令行，进入项目根目录，安装所需的 Python 库。

```bash
pip install -r requirements.txt
```

#### 3\. 运行 Web UI

通过运行主程序文件来启动 Web 界面。程序将自动加载 `config.json` 中的配置地址和端口。

```bash
python app.py
```

启动后，控制台会显示访问地址（默认是 `http://127.0.0.1:7860` 或您在配置中设置的端口）。在浏览器中打开该地址即可开始使用。

-----

### ⚙️ 配置指南

所有关键配置都存储在项目根目录的 `config.json` 文件中。您也可以在 Web UI 的“配置”面板中修改并保存这些设置。

| 键名 | 描述 | 默认值 | 备注 |
| :--- | :--- | :--- | :--- |
| `output_dir` | 文件保存的根目录。 | `"C:/ASMR"` | 强烈建议修改为您的实际路径。 |
| `hq_audio_only` | 是否只下载 HQ (高音质) 音频文件。 | `false` | |
| `max_concurrent_downloads` | 单个作品的最大并发下载连接数。 | `3` | 提高数值可加速下载，但也可能增加服务器压力。 |
| **`proxy`** | **[✨ 新增]** 下载时使用的代理地址。 | `""` (空) | 支持 HTTP/SOCKS。例如：`"http://127.0.0.1:1080"`。 |
| **`listen_host`** | **[✨ 新增]** Web UI 监听的 IP 地址。 | `"127.0.0.1"` | `0.0.0.0` 可用于局域网/公网访问。 |
| **`listen_port`** | **[✨ 新增]** Web UI 监听的端口。 | `7683` | 如果端口被占用，请修改此值 (例如 `8888`)。 |
| `api_url` | 作品信息和搜索的 API 基地址。 | `"https://api.asmr-200.com"` | 一般无需修改。 |

### 📖 使用流程

#### 1\. RJ ID 单作品下载

1.  在“**RJ ID 下载**”标签页，输入 RJ 号 (如 `RJ01234567`)。
2.  点击 **“🔍 获取文件信息”**。作品标题和文件列表将自动填充。
3.  系统将自动选中所有文件索引。
4.  点击 **“🚀 开始下载全部文件”**，在进度条中实时查看下载状态。

#### 2\. 关键词搜索与批量下载

1.  在 **“🔍 关键词搜索”** 标签页，输入关键词（如 `耳语`）。
2.  设置页码和每页数量，点击 **“🔎 搜索作品”**。
3.  搜索结果将展示在表格中。
4.  （可选）点击任一 RJ ID，可联动到下载页查看详情。
5.  点击 **“⬇️ 批量下载列表中所有作品”**，程序将按照列表顺序逐个作品进行下载。
